# candidate-management-system

## Project Description
A full-stack Candidate Management System built using Angular, Spring Boot, and MySQL. It allows users to add, view, edit, delete, and import candidate records.

## Tech Stack
- Frontend: Angular
- Backend: Spring Boot (Java)
- Database: MySQL
- API Testing: Postman

  
## Features
- Add Candidate
- View Candidate List
- Edit Candidate
- Delete Candidate
- CSV Import Functionality



## Project Status 

The application is functionally complete for core candidate management operations. However, a few features are still in progress:

- Edit button click event is pending implementation.
- Success message after delete operation is not yet added.
- CSV import flow is implemented, but imported data is not yet persisted into the database.
- UI styling improvements are still in progress for better responsiveness and consistency.

These are planned for future enhancements and do not affect core CRUD functionality.
